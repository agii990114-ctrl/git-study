# 설치 순서

## uv 설치
```py
powershell -c "irm https://astral.sh/uv/install.ps1 | iex"
```
## 프로젝트 초기화
```py
uv init --app
```

## fastAPI 설치
```py
uv add fastapi --extra standard
```

## fastAPI 코드
```py
from typing import Union

from fastapi import FastAPI

app = FastAPI()


@app.get("/")
def read_root():
    return {"Hello": "World"}


@app.get("/items/{item_id}")
def read_item(item_id: int, q: Union[str, None] = None):
    return {"item_id": item_id, "q": q}
```

## fastAPI 실행
```py
uv run fastapi dev # 개발자모드
uv run fastapi run # 배포모드
```

## 🚀 FastAPI 주요 메소드 데코레이터 정리

| FastAPI 데코레이터 | HTTP 메소드 | CRUD 연산 | 주요 역할 및 용도 |
| :--- | :--- | :--- | :--- |
| **`@app.get()`** | **GET** | **Read** | 데이터를 조회할 때 사용합니다. (예: 게시글 목록 가져오기, 사용자 정보 보기) |
| **`@app.post()`** | **POST** | **Create** | 새로운 리소스를 생성할 때 사용합니다. (예: 회원가입, 게시글 쓰기) |
| **`@app.put()`** | **PUT** | **Update** | 기존 리소스를 전체적으로 수정(교체)할 때 사용합니다. (예: 프로필 정보 전체 수정) |
| **`@app.patch()`** | **PATCH** | **Update** | 리소스의 일부분만 수정할 때 사용합니다. (예: 비밀번호 변경, 제목만 수정) |
| **`@app.delete()`** | **DELETE** | **Delete** | 특정 리소스를 삭제할 때 사용합니다. (예: 게시글 삭제, 계정 탈퇴) |

---

### 💡 추가 정보
* **자동 문서화**: 위 데코레이터를 사용하면 `/docs` (Swagger UI)에서 해당 메소드들이 자동으로 분류되어 나타납니다.
* **데이터 전송**: `GET`은 주로 **Query Parameter**를, `POST/PUT/PATCH`는 **Request Body**(JSON)를 통해 데이터를 전달받습니다.
* **상태 코드**: FastAPI는 각 메소드에 맞춰 기본 응답 상태 코드를 관리하기 매우 쉽습니다. (예: `@app.post(..., status_code=201)`)
