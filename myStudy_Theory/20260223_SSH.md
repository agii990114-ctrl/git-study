## SSH
- 원격제어 프로그램
- 리눅스 필요 (ubuntu 서버를 설치해서 사용)

### dockerfile.Root
```bash
# 1. 베이스 이미지 설정
FROM ubuntu:24.04

# 2. 패키지 업데이트 및 필요한 도구 설치 (SSH, sudo 등)
RUN apt-get update && apt-get install -y \
    openssh-server \
    vim \
    sudo

# 3. SSH 실행을 위한 디렉토리 생성
RUN mkdir /var/run/sshd

# 4. Root 비밀번호 설정 (예: root1234)
RUN echo 'root:root1234' | chpasswd

# 5. Root의 SSH 접속 허용 설정 (sed 명령어로 파일 수정)
RUN sed -i 's/#PermitRootLogin prohibit-password/PermitRootLogin yes/' /etc/ssh/sshd_config

# 6. SSH 포트 개방
EXPOSE 22

# 7. 컨테이너 시작 시 SSH 서비스 자동 실행
CMD ["/usr/sbin/sshd", "-D"]

```

- `docker build -t ubuntu:root -f Dockerfile.Root .`
