# 🐍 파이썬 주요 자료형별 메소드 정리

## 1. 문자열 (String)
문자열은 **불변(Immutable)** 자료형이므로, 메소드 사용 시 원본이 바뀌지 않고 새로운 문자열을 반환합니다.

| 메소드 | 설명 | 예시 |
| :--- | :--- | :--- |
| `.upper()` / `.lower()` | 대문자 / 소문자로 변환 | `"hi".upper()` -> `"HI"` |
| `.strip()` | 양쪽 공백 제거 | `"  hi  ".strip()` -> `"hi"` |
| `.replace(old, new)` | 문자열 치환 | `"apple".replace("p", "b")` -> `"abble"` |
| `.split(sep)` | 특정 구분자로 나누어 리스트 반환 | `"a,b,c".split(",")` -> `['a', 'b', 'c']` |
| `.join(iterable)` | 리스트 등을 하나의 문자열로 합침 | `"-".join(['a', 'b'])` -> `"a-b"` |
| `.find(sub)` / `.index(sub)` | 특정 문자 위치(인덱스) 찾기 | `"abc".find("b")` -> `1` |

---

## 2. 리스트 (List)
리스트는 **가변(Mutable)** 자료형으로, 메소드 호출 시 원본 리스트가 직접 수정되는 경우가 많습니다.

| 메소드 | 설명 | 예시 |
| :--- | :--- | :--- |
| `.append(x)` | 리스트 끝에 요소 추가 | `arr.append(4)` |
| `.extend(list)` | 리스트 뒤에 다른 리스트를 합침 | `arr.extend([5, 6])` |
| `.insert(i, x)` | 특정 위치 `i`에 `x` 삽입 | `arr.insert(0, "start")` |
| `.remove(x)` | 값이 `x`인 첫 번째 요소 삭제 | `arr.remove(3)` |
| `.pop(i)` | `i` 위치의 요소를 꺼내고 삭제 (기본값은 마지막) | `val = arr.pop()` |
| `.sort()` | 리스트 요소를 오름차순 정렬 (원본 수정) | `arr.sort()` |
| `.reverse()` | 리스트 순서를 뒤집음 (원본 수정) | `arr.reverse()` |

---

## 3. 딕셔너리 (Dictionary)
Key-Value 쌍으로 데이터를 관리합니다.

| 메소드 | 설명 | 예시 |
| :--- | :--- | :--- |
| `.keys()` | 모든 Key를 모아서 반환 | `d.keys()` |
| `.values()` | 모든 Value를 모아서 반환 | `d.values()` |
| `.items()` | Key, Value 쌍을 튜플로 반환 | `d.items()` |
| `.get(key, default)` | Key로 값 가져오기 (Key가 없어도 에러 미발생) | `d.get("name", "N/A")` |
| `.update(dict2)` | 다른 딕셔너리 내용을 합치거나 수정 | `d.update({"age": 20})` |
| `.clear()` | 모든 요소 삭제 | `d.clear()` |

---

## 4. 집합 (Set)
중복을 허용하지 않으며 순서가 없습니다.

| 메소드 | 설명 | 예시 |
| :--- | :--- | :--- |
| `.add(x)` | 요소 하나 추가 | `s.add(4)` |
| `.update(list)` | 여러 요소 추가 | `s.update([5, 6])` |
| `.remove(x)` | 특정 요소 삭제 (없으면 에러) | `s.remove(3)` |
| `.discard(x)` | 특정 요소 삭제 (없어도 에러 미발생) | `s.discard(10)` |
| `.intersection(s2)` | 교집합 (`&` 연산자와 동일) | `s1.intersection(s2)` |
| `.union(s2)` | 합집합 (`|` 연산자와 동일) | `s1.union(s2)` |
| `.difference(s2)` | 차집합 (`-` 연산자와 동일) | `s1.difference(s2)` |

---

### 💡 팁
* **`dir(자료형)`**: 해당 자료형이 가진 모든 메소드를 확인
* **`help(객체.메소드)`**: 해당 메소드의 상세 도움말 확인
