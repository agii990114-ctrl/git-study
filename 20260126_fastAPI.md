# 설치 순서

## uv 설치
```py
powershell -c "irm https://astral.sh/uv/install.ps1 | iex"
```
## 프로젝트 초기화
```py
uv init --app
```

## fastAPI 설치
```py
uv add fastapi --extra standard
```

## fastAPI 코드
```py
from typing import Union

from fastapi import FastAPI

app = FastAPI()


@app.get("/")
def read_root():
    return {"Hello": "World"}


@app.get("/items/{item_id}")
def read_item(item_id: int, q: Union[str, None] = None):
    return {"item_id": item_id, "q": q}
```

## fastAPI 실행
```py
uv run fastapi dev # 개발자모드
uv run fastapi run # 배포모드
```

## 데이터 형식

| 메소드 | 역할 | CRUD | 설명 |
| :--- | :--- | :--- | :--- |
| **GET** | 조회 | **Read** | 서버에서 데이터를 가져올 때 사용합니다. URL에 데이터가 노출됩니다. |
| **POST** | 생성 | **Create** | 서버로 데이터를 전송하여 새로운 리소스를 생성합니다. Body에 데이터를 담습니다. |
| **PUT** | 수정 | **Update** | 리소스 전체를 새로운 내용으로 바꿉니다. (전체 교체) |
| **PATCH** | 일부 수정 | **Update** | 리소스의 특정 부분만 수정할 때 사용합니다. (부분 변경) |
| **DELETE** | 삭제 | **Delete** | 지정한 리소스를 삭제합니다. |
